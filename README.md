## How to Use

1. **Open the App:** Allow location permissions so the app can find stations around you.
2. **Browse Stations:** Look at the list or tap icons on the map to see station details.
3. **Start Navigation:** Tap **"Navigate"** on any station card to see your route.
4. **Follow the Path:** Use the blue line and the orientation arrow to guide you.
5. **Arrive:** The app will tell you when you've reached your spot!

## Tech Stack

VoltWay is built with modern Android tools:
- **Language:** Kotlin
- **UI:** Jetpack Compose (Modern native UI)
- **Maps & Navigation:** Mapbox SDK v10 & v2.21.0
- **Location:** Google Play Services Location
